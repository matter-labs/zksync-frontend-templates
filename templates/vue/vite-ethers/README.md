This is a [zkSync](https://zksync.io) + [ethers v6](https://docs.ethers.org/v6/) + [Vite](https://vitejs.dev/) project bootstrapped with [`zksync-cli`](https://github.com/matter-labs/zksync-cli)

# Getting Started

Run `npm run dev` in your terminal, and then open [localhost:3000](http://localhost:3000) in your browser.

Once the webpage has loaded, changes made to files inside the project directory (e.g. `/pages/index.vue`) will automatically update the webpage.
